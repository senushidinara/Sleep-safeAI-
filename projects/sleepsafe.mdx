# SleepSafe

## Overview
SleepSafe is an AI-driven wellness platform designed to help users overcome late-night sleep procrastination—the cycle of staying up too late scrolling, watching, or working. Using **ElevenLabs AI voice technology**, SleepSafe acts as a cognitive co-pilot providing empathetic guidance, real-time interventions, and personalized wellness support.  

## Key Features
- **Dynamic AI Interventions:** Context-aware follow-up questions to explore user feelings.  
- **AI-Powered Session Summary:** Generates concise summaries of user sessions.  
- **Comprehensive Session Export:** Download full transcript, analysis, and personalized plans.  
- **Cognitive Load Monitoring:** Tracks emotional and behavioral patterns with actionable insights.  
- **NeuroSync Audio Lab:** Calming audio routines to replace harmful late-night habits.  

## How It Works
SleepSafe synthesizes user input, behavioral patterns, and AI-detected themes using a **Liquid Cognitive Engine**. It delivers empathetic voice responses, tracks progress, and creates personalized routines to replace destructive sleep procrastination loops with healthy habits.  

## Technologies Used
- **ElevenLabs API:** Text-to-Speech, Voice Cloning for empathetic audio feedback.  
- **Google Gemini API:** AI-driven cognitive analysis.  
- **React & TypeScript:** Frontend framework.  
- **Tailwind CSS:** UI styling.  
- **Node.js & Express:** Backend API proxy for secure integrations.  
- **Vultr:** Cloud hosting.  
- **LiquidMetal AI Raindrop:** Cognitive engine integration.  
- **Web Speech API:** Speech recognition.  

## Getting Started
```bash
# Clone the repository
git clone https://github.com/senushidinara/Sleep-safeAI-.git

# Run the development server
```
